개발 및 운영 담당


  담당업무
- 그룹 인사/교육/총무 시스템 개발 및 운영


자격요건
- 유관 업무 경력 5년 이상 ~ 10년 이하
- Java 및 Spring Framework (Spring Boot 포함) 개발 경험 보유
- RESTful API 설계 및 개발 경험 보유
- RDBMS(MSSQL, MySQL 등) 사용 경험과 SQL 쿼리 작성 능력 보유


필요역량
- 웹 애플리케이션 및 API 설계/개발 역량
- 데이터베이스 관리 및 SQL 활용 능력
- Swagger 및 Postman을 통한 API 문서화 및 테스트 경험
- 기본적인 웹 개발(HTML, CSS, JavaScript) 지식
- 프론트엔드-백엔드 통신에 대한 이해 (비동기 통신 Ajax, Axios, Fetch 경험 등)
- 버전 관리 도구(Git) 사용 능력
- 애플리케이션 성능 최적화 및 문제 해결 능력 (JProfiler 등 성능 분석 도구 사용 경험)
- UI/UX 디자인에 대한 이해와 협업 경험 (Figma 등의 디자인 도구 이해)
- JUnit 등을 활용한 테스트 코드 작성 능력
우대사항
- Front-end 개발 경험 (JavaScript, React 등)
- DevOps 및 CI/CD 경험 (Jenkins, GitLab, GitHub 등)
- Kafka, Redis, Kong API Gateway 활용 경험


제출서류
경력기술서 제출이 필요합니다.









채용절차 

1) 서류전형 합격자를 대상으로 온라인 인성검사를 진행하며(약 30분 소요), 
   면접 시 참고자료로 활용됩니다.
2) 전형 결과는 개별적으로 안내 드립니다. 
3) 상시 채용으로서 채용이 조기에 마감될 수 있습니다.
