- Java Web Programming(Java, JSP) 기술 보유자
- Spring Framework 기술 보유자
- JSTL, XML, JSON 기술 보유자
- Javascript (jQuery, React, Node.JS, Javascript Library)

[우대조건]
- Python , PyTorch 기술역량 우수자 및 프로젝트 수행 유경함자 우대

  :생성형 AI 프롬프트, 파인튜닝, 랭체인 프레임워크 경력자
  :RAG 프로세스 전처리 경력자
- RDB ( MariaDB, MySQL, Oracle ) 지식 보유자 
- 빅데이터분석과정 수료자, 자격증 보유자
