담당 업무
- B2B 웹서비스 솔루션 및 프레임워크 풀스택 개발
- 프론트엔드 위주의 작업을 진행하며 필요할 때는 백엔드 작업도 수행할 수 있음

개발 환경
- 프론트엔드 React, Next.js, TypeScript
- 백엔드 Python (FastAPI), Java (Springboot)

- React, Next.js를 이용한 서비스 개발 경험이 있으신 분
- Amazon Web Service 또는 Naver Cloud Platform을 사용한 서비스 경험이 있는 분

- Python, Java 언어 가능자

- 컴퓨터 관련 학과 전공자거나 웹서비스 관련 CS 지식이 풍부하신 분
- 원활한 커뮤니케이션 능력이 있으신분
